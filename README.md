# react-native-haptic

React Native module for triggering Android haptic feedback

## Installation

```sh
npm install react-native-haptic
```

## Usage


```js
import {triggerVirtualKeyFeedback} from 'react-native-feedback-haptics';

// ...

triggerVirtualKeyFeedback();
```
